# Roblox Chat Platform

A **safe multiplayer chat platform for Roblox players**, designed to allow real-time messaging, friend requests, and theme customization. Built with Node.js, React, MongoDB, and Socket.IO. Perfect for users under 13 who can no longer use Roblox’s native chat system.

---

## Features

- ✅ Signup/Login with unique username
- ✅ Add friends via username
- ✅ Friend requests and acceptance system
- ✅ Real-time chat with Socket.IO
- ✅ Chat history saved in MongoDB
- ✅ Green theme with switcher to yellow theme
- ✅ Responsive, fun, and modern UI

---

## Tech Stack

- **Backend:** Node.js, Express, MongoDB, Socket.IO
- **Frontend:** React.js
- **Deployment:** Render, Docker-ready
- **Authentication:** JWT + bcrypt
